This is a full-stack web application that **parses data from a given GitHub profile** and uses an **AI Chat** to generate a **pdf resume**.

## üöÄ Features

- üîó **GitHub Integration**: Input a GitHub link and extract relevant user data (repos, technologies, languages).
- üßæ **Parsing Service**: A backend service that parses and transforms GitHub data into structured format for resume generation.
- üí¨ **AI Chat Service**: A backend service that receives formatted user data and generates latex code for a resume.
- üñ•Ô∏è **Frontend**: Fancy UI to input GitHub URL, interact with AI, and view LaTeX output.
- üîô **Backend API**: Handles routing, logic, latext to pdf compilation, and inter-service communication.
- üóÇÔ∏è **Database**: Stores account data.
